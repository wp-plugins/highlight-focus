Highlight Focus on tab key pressed.

== Description ==
Highlight Focused Element on tab key pressed so you can easely follow it.

Features :

1. Setup effect duration in ms
2. Setup hide delay in ms
3. Choose background and border color of the hilighted element
4. Setup background and border opacity of the hilighted element 
5. Show or hide the default focus outline (hiding it break accessibility rules)

Visit [JuZED.fr](http://www.juzed.fr/) for example (just press "tab" key and follow the focus)

== Installation ==
1. Upload "highlight-focus.zip" to the "/wp-content/plugins/" directory.
2. Unzip archive.
3. Activate the plugin through the "Plugins" menu in WordPress.
4. Save settings in "Settings / Hilight Focus" at least once to create css file (no need when upgrade)

== Frequently Asked Questions ==
= Is Highlight Focus accessible ? =
Yes, it has been made for this purpose (but it can be unvalidate if you check "Hide regular outline" option).
It follows the WCAG (Web Content Accessibility Guidelines) to at least level 'A'.

= If my website is not accessible, does it make it accessible ? =
No, your site has to be accessible so it can keep his accessibility level when the plugin is added.
